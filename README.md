# AgeCalculator Pro

Free age calculator. No login or installation needed. Works in any browser.

## Features
- Exact age in years, months, days, hours and minutes
- Live updating timer (updates every second)
- Day of the week you were born
- Days until next birthday with progress bar
- Fun facts — heartbeats, hours slept, birthdays
- Compare ages between two people
- Shareable age card
- Dark and light mode
